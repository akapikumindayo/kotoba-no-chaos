# kotoba-no-chaos
文章を指定した単語で指定した順番にランダムに作る笑えるツールです。

プレイはこちらから
(https://akapikumindayo.github.io/kotoba-no-chaos/)

右側で各品詞に単語を追加、削除できます。
左側一番上で品詞の順を指定できます。
その下で品詞(グループ)の追加ができます。
さらにその下で生成ができ、読み上げ、コピー、ハート保存もできます。
※実際に流れている音声は権利上の問題がある可能性があるため、インターネットに流すことはご遠慮ください。
さらにさらにその下では各テーマのプリセットを選択できます。

その他の機能として辞書をJSON保存してほかの人に配布することや、バックアップを取ることができます。
自動セーブ機能はあります。(ブラウザ保存)

お気に入り「ピチピチなお掃除ロボットは炙りサーモン寿司が優雅に即興ポエムを詠むのである。」
